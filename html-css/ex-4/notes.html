Notes: CSS Units and Responsive Design

Introduction

I learned about different CSS units and responsive techniques to create scalable and responsive designs.

Key Takeaways

  • CSS units like px, em, rem, ch, vh, and vw can be used to create responsive designs.
  • min(), max(), and clamp() functions can be used to create scalable font sizes and widths.
  • Continuous responsiveness can be achieved by avoiding fixed breakpoints and using flexible grids and fluid typography.

CSS Techniques Used

  • CSS units (px, em, rem, ch, vh, vw)
  • min(), max(), and clamp() functions
  • Flexible grids and fluid typography
  • Continuous responsiveness
