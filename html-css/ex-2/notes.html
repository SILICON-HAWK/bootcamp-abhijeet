Notes: CSS Variables and Basic Styling

Introduction

I learned about the cascading nature of styles, how to override and customize styles, and how to theme styles using light and dark modes.

Key Takeaways

  • CSS variables help manage consistent styling across a webpage.
  • I can define and use variables for colors, fonts, and spacings.
  • Default values can be set for variables when they are not defined.
  • CSS selectors can be used to style alternate rows of a table.
  • Variables can be used to define other variables.
  • A dynamic theme switcher can be created using variables to toggle between light and dark modes.

CSS Techniques Used

  • CSS variables (custom properties)
  • Default values for variables
  • CSS selectors (nth-child)
  • Dynamic theme switching using variables
