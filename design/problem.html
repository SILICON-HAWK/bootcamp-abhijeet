Design Principles and Exercises


Fonts

Introduction to fonts, including different font types, Google Fonts, and local font installation.

  • Understand different font types (serif, sans-serif, monospace, etc.)
  • Learn how to use Google Fonts and install fonts locally
  • Explore how font choices impact readability and aesthetics

Exercise 1: Basics of Fonts

Level 1:

  • Use the same webpage
  • Add a snippet of JS code (10 lines) to the HTML
  • Apply basic font styles using system fonts (e.g., Arial, Times New Roman, Courier)
  • Change the weights and apply different weight to different classes

Level 2:

  • Use Google Fonts to apply three fonts (sans-serif for text, serif for headings, monospace for code)
  • Check how variable fonts work
  • Illustrate it in your page

Level 3:

  • Install a custom font locally and use it in the page
  • Combine multiple font families to create a typographic hierarchy

Colors

Introduction to color theory, CSS color formats, and color usage.

  • Learn about color theory: color wheels, themes, and combinations
  • Understand CSS color formats (hex, RGB, HSL) and alpha values
  • Explore the use of shadows, borders, and transparency

Exercise 2: Basics of Colors

Level 1:

  • Add a card to the page
  • Apply a monochromatic theme to the entire page using a base color
  • Ensure the card (background, border, and shadow) stands out using alpha channels for subtle differentiation

Level 2:

  • Create dark and light themes for the page using CSS variables
  • Define --primary, --secondary, and --background colors
  • Create two separate stylesheets, one for dark and one for light mode

Level 3:

  • Design a gradient theme using only CSS
  • Apply linear gradients to headers and radial gradients to buttons

Icons

Introduction to icon types, usage, and implementation.

  • Learn about different icon types: SVGs and icon fonts
  • Explore use cases for icons in navigation, buttons, and content
  • Compare advantages and limitations of SVGs and fonts

Exercise 3: Basics of Icons

Level 1:

  • Add three icons to a page using an icon font library (e.g., Font Awesome)
  • Style the icons using CSS (e.g., size, color)

Level 2:

  • Replace one icon with an SVG file
  • Create a grid of six icons (three from a font library and three as inline SVG)
  • Add hover effects to icons (e.g., color change, scaling)

Level 3:

  • Design a navigation bar with icons as the main interactive elements
  • Include home, about, and contact icons
  • Align the icons with labels using CSS
  • Create buttons for download and upload, embedding SVG icons

Images

Introduction to image types, editing, and implementation.

  • Understand image types (JPEG, PNG, SVG) and their use cases
  • Learn basic image editing: transparency, compression, and resizing
  • Explore advanced techniques: full-bleed images, placeholders, and responsive images

Exercise 4: Basics of Images

Level 1:

  • Add three images to a web page
  • Apply CSS styles for width, height, and border-radius
  • Use a placeholder image service (e.g., Lorem Picsum)

Level 2:

  • Edit an image to adjust its size and compression
  • Add a full-bleed image as a page header
  • Use CSS to add overlays (e.g., gradient or color tint)

Level 3:

  • Implement responsive images using the element
  • Create an image with high-res and low-res versions for different devices
  • Optimize image loading using lazy loading techniques

Layouts and Spacing

Introduction to creating balanced layouts using CSS.

  • Learn to create balanced layouts using CSS
  • Explore spacing techniques: margins, padding, and grid systems
  • Understand the importance of alignment and consistency

Exercise 5: Layout Principles

Level 1:

  • Recreate the layout of a simple webpage (e.g., medium.com article)
  • Focus on applying consistent margins and padding
  • Use white space to create clear separation between sections
  • Align text and images correctly

Level 2:

  • Demonstrate a layout that emphasizes visual hierarchy
  • Design a homepage with a large hero section, a featured article grid, and a footer
  • Use larger fonts and bolder colors for important elements
  • Add gutters between sections to maintain visual clarity
  • Apply a CSS grid system to organize content effectively

Design Principles in Action

Review and critique of Indian websites.

  • Review a few Indian websites and identify areas for improvement
  • Critique the websites based on design principles and suggest improvements

Exercise 6: Redesign of Websites

Level 1:

  • For each of the websites given, annotate the problems with it using the screenshot tool mentioned
  • Suggest websites that have done similar things better, that can be used for inspiration

Level 2:

  • Compare two websites for both design and information structure
  • Design a navigation bar with icons as the main interactive elements

Level 3:

  • Do redesign of at least two of the sites from level 1 exercises
  • Feel free to cut out parts of the website and implement only the relevant part of the website

Websites to review:

  • Indian Government Press Releases (Hint: take a look at US Department of State)
  • IITM CSE Faculty site
  • IIT B Talk
  • Indian Railways website
